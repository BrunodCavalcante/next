NEXT AGROLINK - PROTÓTIPO FUNCIONAL REVISADO

Como usar:
1. Extraia o ZIP.
2. Abra o arquivo index.html no navegador.
3. Não precisa instalar nada.

Esta versão foi refeita como uma aplicação HTML/CSS/JS de página única (SPA), para evitar telas quebradas e links desconfigurados.

Funcionalidades simuladas:
- Menu lateral e menu mobile funcionando
- Navegação entre todos os módulos
- Dashboard com indicadores e gráfico em CSS
- Lista de clientes com ver, editar, excluir e novo cadastro
- Lista de produtos com ver, editar, excluir e cadastro completo por abas
- Lista de pedidos com ver, editar, excluir e novo pedido
- Vendas e comissões calculadas a partir dos dados do protótipo
- Módulos de estoque, produção, qualidade, financeiro, logística, BI, auditoria e portal representante
- Modais funcionais
- Toasts de confirmação
- Responsivo para celular

Observação:
É uma versão sem banco de dados. As alterações funcionam durante o uso da página, mas ao recarregar o navegador os dados voltam ao padrão inicial.
